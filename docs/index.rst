Python-Arango
-------------

Welcome to the documentation for **python-arango**, a Python driver for ArangoDB_.

Requirements
=============

- ArangoDB version 3.9+
- Python version 3.8+

Installation
============

.. code-block:: bash

    ~$ pip install python-arango --upgrade
